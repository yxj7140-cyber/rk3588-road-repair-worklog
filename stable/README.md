# Stable Deliverables

This folder is for mature, restore-ready project files.

Current expected stable areas:

- `chassis/`: formal chassis migration logic.
- `web-remote/`: browser remote control service, safe-lock first.
- `restore-0618/`: exact files and instructions needed to restore the 2026-06-18 state.
- `hyperboot/`: selected RT image metadata and small helper files.

Large binaries should usually be represented by metadata and hashes, not committed directly.
